# 최신 논문 동향 분석 대시보드
- Computer Science 분야의 최근 출간 논문들에 대한 요약 정보와 트렌드 파악을 지원하는 통합 대시보드 플랫폼
--
## 🌵papescope 폴더구조
```
├─Analysis       # 전처리 코드 또는 분석용 테이블 생성 쿼리
├─Dashboard      # 대시보드
└─ETL            # raw data 추출, 변환 및 저장 코드
    ├─csv        # 추출된 csv 데이터
    ├─extract    # API 호출 코드
    └─transform  # json to csv 변환 코드
```
